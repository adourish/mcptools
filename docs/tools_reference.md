## Mermaid to Visio Converter

**Location:** `G:\My Drive\06_Skills\documentation\diagram-tools\`

**Purpose:** Convert Mermaid diagrams from markdown files to Visio-compatible formats (SVG, PNG, PDF)

**Files:**
- `mermaid_to_visio.py` - Main conversion tool
- `README.md` - Complete documentation
- `QUICKSTART.md` - Quick start guide
- `test_diagrams.md` - Sample diagrams for testing

**Quick Usage:**
```bash
# Navigate to the tool
cd "G:\My Drive\06_Skills\documentation\diagram-tools"

# Convert diagrams
python mermaid_to_visio.py your_file.md --formats svg
```

**Documentation:** See README.md in the tool folder for complete guide.
